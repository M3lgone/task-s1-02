# Sprint 1 — Task 02: PHP Basics

## 📄 Description

This repository contains the exercises for **Sprint 1 — Task 02**.

In this task, the fundamentals of **PHP** were practiced, including variables, conditionals, loops, functions, constants, and basic operations.

---

## 🎯 Objectives

* Create and use variables and constants
* Perform operations with different data types
* Practice conditional statements and loops
* Organize code using functions

---

## 🛠 Technologies

* PHP

---

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/M3lgone/task-s1-02.git
```

---

## ▶️ How to Run

Run each level independently using PHP's built-in server.

### Example:

```bash
php -S localhost:8000 -t level-1
```

Then open:

```
http://localhost:8000
```

👉 Replace `level-1` with `level-2` or `level-3` depending on what you want to test.
